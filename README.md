# optimistic rabbit server

API server for optimistic-rabbit-web
is school project for Unicorn University

## Installation

Use the package manager [yarn](https://pip.pypa.io/en/stable/) to install foobar.
```bash
yarn add
```

## Stack

 - Prisma as ORM
 - PlanerScale for database
 - Express / Node.js
